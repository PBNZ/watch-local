# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, assume good faith, no harassment of any kind.
Disagreements about code are fine; personal attacks are not.

Report unacceptable behavior by opening a GitHub issue on this repository
(say it concerns conduct and a private channel will be arranged, same as
the security-reporting flow in SECURITY.md). The maintainer will review
and respond as promptly as reasonably possible, and may edit, remove, or
reject contributions and interactions that violate this code.
